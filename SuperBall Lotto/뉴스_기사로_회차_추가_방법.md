# 뉴스 기사로 당첨 회차 추가하는 방법

동행복권 접속이 막혀 자동 수집이 안 될 때, **해당 회차 당첨번호가 나온 뉴스 기사 URL**을 쓰면 그 회차만 캐시에 추가할 수 있습니다.

---

## 1단계: 회차 정하기

- 보고 싶은 회차 번호를 정합니다. (예: 501회, 1214회)

---

## 2단계: 뉴스 기사 찾기

1. **검색**  
   네이버·구글 등에서 다음처럼 검색합니다.  
   - `501회 로또 당첨번호`  
   - `로또 1214회 당첨`

2. **기사 선택**  
   제목에 **"N회 당첨번호"**, **"당첨번호"**, **"보너스"** 같은 말이 들어간 **뉴스 기사**를 엽니다.  
   - 예: "로또 1214회 당첨번호 '10·15·19·27·30·33'…1등 12명"  
   - 지역신문, 연합뉴스, 스포츠/경제 뉴스 등 대부분 괜찮습니다.

3. **내용 확인**  
   기사 본문에 아래가 **한 군데 이상** 있으면 됩니다.  
   - **회차**: "제501회", "1214회", "로또 501회" 등  
   - **당첨번호 6개**: '10·15·19·27·30·33' 또는 "10, 15, 19, 27, 30, 33" 형태  
   - **보너스**: "보너스 번호 '14'", "2등 보너스 14" 등  

4. **주소 복사**  
   브라우저 **주소창**의 URL을 복사합니다.  
   - 예: `https://www.kado.net/news/articleView.html?idxno=2037426`

---

## 3단계: 웹 페이지에서 회차 추가

1. **로또 확률 번호 생성기** 페이지를 엽니다.  
   - 주소: http://localhost:5000 (서버 실행 후)

2. **「1등 당첨 기록 수집 (수동)」** 카드로 내려갑니다.

3. **「뉴스 기사 URL에서 회차 추가」** 입력칸에  
   복사한 **기사 URL**을 붙여넣습니다.

4. **「기사 URL에서 추가」** 버튼을 누릅니다.

5. **결과 확인**  
   - 성공: "제501회 추가됨. 총 N회차 보유." 같은 문구가 나옵니다.  
   - 실패: "기사에서 당첨번호를 찾지 못했습니다" → 다른 기사로 같은 회차를 다시 시도해 보세요.

---

## 4단계: 추가된 회차 보기

- **「당첨 기록 (웹 표시)」** 카드에서  
  **「회차 지정하여 보기」** 드롭다운을 눌러 방금 추가한 회차(예: 501회)를 선택하면  
  당첨번호·보너스가 표시됩니다.
- 아래 **최근 회차 목록**에도 같은 회차가 보입니다.

---

## 팁

- **여러 회차**를 쓰려면: 회차마다 위 1~3단계를 반복해서 기사 URL을 하나씩 추가하면 됩니다.
- **같은 회차를 여러 번 추가**해도 캐시에는 한 번만 저장됩니다 (덮어쓰기).
- 기사 제목/본문에 **회차 숫자**와 **당첨번호 6개**, **보너스**가 있으면 대부분 인식됩니다.

---

## CMD에서 URL로 추가하기

웹이 아니라 터미널에서 하고 싶다면:

```bash
cd "g:\내 드라이브\KNOU\Somoim\Jungwon_Drive_Obsidian_Vault\SuperBall Lotto"
python main.py add_url --url "여기에_뉴스_기사_URL_붙여넣기"
```

예:

```bash
python main.py add_url --url "https://www.kado.net/news/articleView.html?idxno=2037426"
```

---

이 방법으로 추가한 회차는 **확률 통계**와 **번호 생성**에도 반영됩니다.
