# 로그인 버튼 숨김 작업 기록

## 작업 목적
사용자 혼란 방지를 위해 현재는 구글 로그인만 표시하고, 나머지 로그인 옵션들은 백그라운드에서 개발 중

## 작업 내용

### 1. CSS 수정 사항
- **파일**: `LatinDance_Party/static/css/style.css`
- **수정 위치**: 소셜 로그인 버튼 스타일 섹션

### 2. 변경된 스타일

#### 2.1 소셜 버튼 그리드 레이아웃 변경
```css
.social-buttons {
    display: grid;
    grid-template-columns: 1fr; /* 2열에서 1열로 변경 */
    gap: 10px;
    margin-bottom: 15px;
}
```

#### 2.2 숨겨진 로그인 버튼들
```css
/* 다른 로그인 버튼들 숨기기 */
.kakao-login-btn,
.naver-login-btn,
.apple-login-btn {
    display: none !important;
}

/* 이메일 로그인 섹션 숨기기 */
.email-login-section {
    display: none !important;
}

/* 전화번호 로그인 섹션 숨기기 */
.phone-login-section {
    display: none !important;
}

/* 게스트 로그인 섹션 숨기기 */
.guest-login-section {
    display: none !important;
}

/* 구분선 숨기기 */
.login-divider {
    display: none !important;
}
```

## 현재 상태
- ✅ **표시됨**: 구글 로그인 버튼만
- ❌ **숨겨짐**: 카카오톡, 네이버, 애플 로그인
- ❌ **숨겨짐**: 이메일 로그인
- ❌ **숨겨짐**: 전화번호 로그인
- ❌ **숨겨짐**: 게스트 로그인

## 향후 계획
1. 각 로그인 방식별로 백그라운드에서 개발 완료
2. 충분한 테스트 후 점진적으로 표시
3. 사용자 피드백 반영하여 최종 배포
