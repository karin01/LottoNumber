# 카카오톡 로그인 연동 가이드

## 📋 필요 단계

### 1. 카카오 개발자 계정 설정
1. **카카오 개발자 사이트 접속**
   - https://developers.kakao.com 접속
   - 카카오 계정으로 로그인

2. **애플리케이션 생성**
   - "내 애플리케이션" → "애플리케이션 추가하기"
   - 앱 이름: "라틴댄스 파티" (또는 원하는 이름)
   - 회사명: 본인 이름 또는 회사명

3. **플랫폼 설정**
   - "플랫폼" → "Web" 플랫폼 등록
   - 사이트 도메인: `https://latinmat.co.kr`
   - 개발용 도메인: `http://localhost:5000`

4. **카카오 로그인 활성화**
   - "카카오 로그인" → "활성화 설정"
   - "Redirect URI" 설정:
     - `https://latinmat.co.kr/auth/kakao/callback`
     - `http://localhost:5000/auth/kakao/callback`

5. **JavaScript 키 확인**
   - "앱 키" 메뉴에서 JavaScript 키 복사
   - 예: `1234567890abcdef1234567890abcdef`

### 2. Firebase 설정

#### A. Firebase Console에서 설정
1. **Firebase Console 접속**
   - https://console.firebase.google.com
   - 프로젝트 선택

2. **Authentication → Sign-in method**
   - "카카오" 제공업체 추가
   - Client ID: 카카오 JavaScript 키 입력
   - Client Secret: 카카오 Secret 키 입력

#### B. Firebase Functions 설정 (선택사항)
```javascript
// functions/index.js
const functions = require('firebase-functions');
const admin = require('firebase-admin');

exports.kakaoAuth = functions.https.onCall(async (data, context) => {
  // 카카오 토큰 검증 및 Firebase 커스텀 토큰 생성
  const kakaoToken = data.kakaoToken;
  
  // 카카오 API로 사용자 정보 조회
  const userInfo = await getKakaoUserInfo(kakaoToken);
  
  // Firebase 커스텀 토큰 생성
  const customToken = await admin.auth().createCustomToken(userInfo.id);
  
  return { customToken };
});
```

// main.js의 signInWithKakao 함수 수정
function signInWithKakao() {
    // 카카오 SDK 초기화
    if (!Kakao.isInitialized()) {
        Kakao.init('YOUR_JAVASCRIPT_KEY'); // 카카오 JavaScript 키
    }
    
    // 카카오 로그인 실행
    Kakao.Auth.login({
        success: function(authObj) {
            console.log('카카오 로그인 성공:', authObj);
            
            // 카카오 사용자 정보 조회
            Kakao.API.request({
                url: '/v2/user/me',
                success: function(res) {
                    console.log('카카오 사용자 정보:', res);
                    
                    // Firebase 커스텀 토큰으로 로그인
                    signInWithKakaoCustomToken(authObj.access_token);
                },
                fail: function(error) {
                    console.error('카카오 사용자 정보 조회 실패:', error);
                    showToast('카카오 로그인에 실패했습니다.', 'error');
                }
            });
        },
        fail: function(err) {
            console.error('카카오 로그인 실패:', err);
            showToast('카카오 로그인에 실패했습니다.', 'error');
        }
    });
}

### 4. 백엔드 설정 (선택사항)

#### A. Flask 백엔드에 카카오 콜백 엔드포인트 추가
```python
# app.py에 추가
from flask import request, jsonify
import requests

@app.route('/auth/kakao/callback')
def kakao_callback():
    code = request.args.get('code')
    
    # 카카오 토큰 교환
    token_url = 'https://kauth.kakao.com/oauth/token'
    token_data = {
        'grant_type': 'authorization_code',
        'client_id': 'YOUR_JAVASCRIPT_KEY',
        'redirect_uri': 'https://latinmat.co.kr/auth/kakao/callback',
        'code': code
    }
    
    token_response = requests.post(token_url, data=token_data)
    token_info = token_response.json()
    
    # 사용자 정보 조회
    user_url = 'https://kapi.kakao.com/v2/user/me'
    headers = {'Authorization': f"Bearer {token_info['access_token']}"}
    user_response = requests.get(user_url, headers=headers)
    user_info = user_response.json()
    
    # Firebase 커스텀 토큰 생성 및 반환
    # ... Firebase Admin SDK 사용
    
    return jsonify({'success': True, 'user': user_info})
```
