# 🕺💃 라틴댄스 파티 홍보 웹 앱

라틴댄스 파티 포스터와 정보를 공유할 수 있는 현대적인 웹 애플리케이션입니다.

## ✨ 주요 기능

- **메뉴 시스템**: 파티 등록하기, 등록된 파티, 라틴 댄스 유튜브 섹션 전환
- **파티 등록**: 제목, 지역, 장소, 날짜, 시간, 설명, 연락처 등록
- **포스터 업로드**: 파티 포스터 이미지 업로드 및 미리보기
- **파티 목록**: 등록된 파티들을 카드 형태로 표시
- **필터링**: 지역별, 날짜별 파티 검색
- **파티 수정/삭제**: 등록된 파티 편집 및 삭제 기능
- **갤러리 시스템**: 파티 관련 추가 사진 업로드 및 갤러리 표시
- **좋아요 시스템**: 파티에 좋아요 표시 및 카운트 기능
- **댓글 시스템**: 파티별 댓글 작성 및 표시
- **상세 모달**: 파티 상세 정보, 갤러리, 댓글을 한 번에 볼 수 있는 모달
- **유튜브 영상 리스트**: 라틴댄스 학습 영상 추천 및 재생
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기 지원
- **드래그 앤 드롭**: 포스터 이미지 드래그 앤 드롭 업로드
- **실시간 유효성 검사**: 입력 데이터 검증 및 에러 메시지 표시

## 🚀 빠른 시작

### 1. 파일 다운로드
```bash
# 프로젝트 폴더를 웹 서버에 업로드하거나 로컬에서 실행
```

### 2. 로컬 실행 (개발 모드)
- `index.html` 파일을 브라우저에서 열기
- 개발 모드에서는 로컬 스토리지를 사용하여 데이터 저장
- Firebase 설정 없이도 모든 기능 테스트 가능

### 3. Firebase 설정 (실제 배포용)
1. [Firebase 콘솔](https://console.firebase.google.com/) 에서 새 프로젝트 생성
2. Firestore Database 활성화
3. Storage 활성화
4. `js/firebase-config.js` 파일의 설정 업데이트:

```javascript
const firebaseConfig = {
    apiKey: "your-api-key",
    authDomain: "your-project.firebaseapp.com",
    projectId: "your-project-id",
    storageBucket: "your-project.appspot.com",
    messagingSenderId: "your-sender-id",
    appId: "your-app-id"
};
```

### 4. Firestore 보안 규칙 설정
```javascript
// Firestore 규칙 예시
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /parties/{document} {
      allow read, write: if true; // 필요에 따라 보안 규칙 수정
    }
  }
}
```

### 5. Storage 보안 규칙 설정
```javascript
// Storage 규칙 예시
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /party-posters/{allPaths=**} {
      allow read, write: if true; // 필요에 따라 보안 규칙 수정
    }
  }
}
```

## 💡 사용 방법

### 파티 등록하기
1. "파티 등록하기" 섹션에서 필요한 정보 입력
2. 포스터 이미지 업로드 (선택사항)
3. "파티 등록하기" 버튼 클릭

### 파티 관리하기
1. 파티 카드에서 "수정" 버튼 클릭하여 파티 정보 수정
2. "삭제" 버튼으로 파티 삭제 (확인 후 실행)
3. 수정 시 기존 정보가 폼에 자동으로 채워짐

### 파티 상호작용하기
1. **좋아요**: 파티 카드의 "좋아요" 버튼 클릭
2. **상세보기**: "상세보기" 버튼으로 파티 상세 정보 확인
3. **갤러리**: 상세 모달에서 파티 관련 사진 추가 업로드 및 확인
4. **댓글**: 상세 모달에서 댓글 작성 및 다른 사람들과 소통

### 파티 검색하기
1. "등록된 파티 목록" 섹션에서 필터 사용
2. 지역별 또는 날짜별 검색 가능
3. 파티 카드를 통해 기본 정보 확인 후 상세보기 클릭

### 포스터 및 갤러리 업로드
- **포스터**: 파티 등록 시 메인 포스터 업로드
- **갤러리**: 파티 상세 모달에서 여러 사진 업로드 가능
- 파일 선택 버튼 클릭 또는 드래그 앤 드롭으로 업로드
- 지원 형식: JPG, PNG, GIF
- 최대 파일 크기: 5MB

## 🐛 문제 해결

### 일반적인 문제들

1. **Firebase 초기화 오류**
   - 설정 정보 확인
   - 프로젝트 권한 확인

2. **이미지 업로드 실패**
   - 파일 크기 확인 (5MB 이하)
   - 파일 형식 확인 (이미지만 허용)

3. **파티 목록이 표시되지 않음**
   - 네트워크 연결 확인
   - 브라우저 콘솔 오류 확인
