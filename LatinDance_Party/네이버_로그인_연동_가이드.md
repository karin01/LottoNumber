# 네이버 로그인 연동 가이드

## 📋 필요 단계

### 1. 네이버 개발자 센터 설정
1. **네이버 개발자 센터 접속**
   - https://developers.naver.com 접속
   - 네이버 계정으로 로그인

2. **애플리케이션 등록**
   - "Application → 애플리케이션 등록"
   - 애플리케이션 이름: "라틴댄스 파티"
   - 사용 API: "로그인 오픈 API 서비스 환경"
   - 비즈니스 모델: "개인"

3. **환경 설정**
   - "환경 추가" → "Web 서비스 URL"
   - 서비스 URL: `https://latinmat.co.kr`
   - 개발용 URL: `http://localhost:5000`

4. **로그인 오픈 API 서비스 환경**
   - "로그인 오픈 API 서비스 환경" 추가
   - 서비스 URL: `https://latinmat.co.kr`
   - Callback URL: `https://latinmat.co.kr/auth/naver/callback`
   - 개발용 Callback URL: `http://localhost:5000/auth/naver/callback`

5. **클라이언트 정보 확인**
   - "애플리케이션 정보"에서 확인
   - Client ID (클라이언트 아이디)
   - Client Secret (클라이언트 시크릿)

### 2. Firebase 설정

#### A. Firebase Console에서 설정
1. **Firebase Console 접속**
   - https://console.firebase.google.com
   - 프로젝트 선택

2. **Authentication → Sign-in method**
   - "네이버" 제공업체 추가 (직접 지원하지 않을 수 있음)
   - 커스텀 제공업체로 설정 필요

#### B. Firebase Functions 설정
```javascript
// functions/index.js
const functions = require('firebase-functions');
const admin = require('firebase-admin');
const axios = require('axios');

exports.naverAuth = functions.https.onCall(async (data, context) => {
  const { code, state } = data;
  
  try {
    // 네이버 토큰 교환
    const tokenResponse = await axios.post('https://nid.naver.com/oauth2.0/token', {
      grant_type: 'authorization_code',
      client_id: process.env.NAVER_CLIENT_ID,
      client_secret: process.env.NAVER_CLIENT_SECRET,
      code: code,
      state: state
    });
    
    const { access_token } = tokenResponse.data;
    
    // 네이버 사용자 정보 조회
    const userResponse = await axios.get('https://openapi.naver.com/v1/nid/me', {
      headers: {
        'Authorization': `Bearer ${access_token}`
      }
    });
    
    const userInfo = userResponse.data.response;
    
    // Firebase 커스텀 토큰 생성
    const customToken = await admin.auth().createCustomToken(userInfo.id, {
      provider: 'naver',
      email: userInfo.email,
      displayName: userInfo.name
    });
    
    return { customToken, userInfo };
  } catch (error) {
    throw new functions.https.HttpsError('internal', '네이버 로그인 실패');
  }
});
```

```javascript
// main.js의 signInWithNaver 함수 수정
function signInWithNaver() {
    // 네이버 로그인 팝업 열기
    const naverLoginUrl = `https://nid.naver.com/oauth2.0/authorize?response_type=code&client_id=${NAVER_CLIENT_ID}&redirect_uri=${encodeURIComponent('https://latinmat.co.kr/auth/naver/callback')}&state=${generateRandomState()}`;
    
    const popup = window.open(naverLoginUrl, 'naverLogin', 'width=500,height=600');
    
    // 팝업에서 메시지 수신
    window.addEventListener('message', async function(event) {
        if (event.origin !== 'https://latinmat.co.kr') return;
        
        if (event.data.type === 'NAVER_LOGIN_SUCCESS') {
            const { code, state } = event.data;
            
            try {
                // Firebase Functions 호출
                const result = await firebase.functions().httpsCallable('naverAuth')({ code, state });
                
                // Firebase 커스텀 토큰으로 로그인
                await auth.signInWithCustomToken(result.data.customToken);
                
                closeLoginModal();
                showToast('네이버로 로그인되었습니다!', 'success');
            } catch (error) {
                console.error('네이버 로그인 실패:', error);
                showToast('네이버 로그인에 실패했습니다.', 'error');
            }
        }
    });
}

// 랜덤 state 생성
function generateRandomState() {
    return Math.random().toString(36).substring(2, 15);
}
```

### 4. 백엔드 설정

#### A. Flask 백엔드에 네이버 콜백 엔드포인트 추가
```python
# app.py에 추가
from flask import request, jsonify
import requests
import os

@app.route('/auth/naver/callback')
def naver_callback():
    code = request.args.get('code')
    state = request.args.get('state')
    
    # 네이버 토큰 교환
    token_url = 'https://nid.naver.com/oauth2.0/token'
    token_data = {
        'grant_type': 'authorization_code',
        'client_id': os.environ.get('NAVER_CLIENT_ID'),
        'client_secret': os.environ.get('NAVER_CLIENT_SECRET'),
        'code': code,
        'state': state
    }
    
    token_response = requests.post(token_url, data=token_data)
    token_info = token_response.json()
    
    if 'access_token' not in token_info:
        return jsonify({'error': '토큰 교환 실패'}), 400
    
    # 네이버 사용자 정보 조회
    user_url = 'https://openapi.naver.com/v1/nid/me'
    headers = {'Authorization': f"Bearer {token_info['access_token']}"}
    user_response = requests.get(user_url, headers=headers)
    user_info = user_response.json()
    
    # Firebase 커스텀 토큰 생성
    # Firebase Admin SDK 사용하여 커스텀 토큰 생성
    
    return jsonify({
        'success': True,
        'user': user_info['response'],
        'customToken': custom_token
    })
```
