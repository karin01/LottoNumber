# 🎯 Google AdSense 도메인 설정 문제 해결 가이드

## 🚨 **현재 문제 상황**

- **현재 호스팅**: Firebase (`share-note-ef791.web.app`)
- **애드센스 도메인**: `latinmat.co.kr`
- **문제**: 도메인 불일치로 인한 사이트 확인 실패

## 🔧 **해결 방법**

### **방법 1: 커스텀 도메인 설정 (권장)**

#### **1단계: 도메인 구매 및 설정**
1. **도메인 구매**: `latinmat.co.kr` 도메인을 구매
2. **DNS 설정**: 도메인의 DNS를 Firebase로 연결

#### **2단계: Firebase 커스텀 도메인 설정**
```bash
# Firebase CLI로 커스텀 도메인 추가
firebase hosting:sites:add latinmat
firebase target:apply hosting latinmat latinmat.co.kr
```

#### **3단계: firebase.json 설정**
```json
{
  "hosting": {
    "target": "latinmat",
    "public": ".",
    "ignore": ["firebase.json", "**/.*", "**/node_modules/**"]
  }
}
```

### **방법 2: 임시 해결책 (개발용)**

#### **현재 Firebase 도메인으로 애드센스 설정**
1. **애드센스 계정에서 도메인 변경**:
   - `latinmat.co.kr` → `share-note-ef791.web.app`
2. **코드 재삽입 후 확인**

### **방법 3: 무료 도메인 사용**

#### **GitHub Pages + 커스텀 도메인**
1. **GitHub Pages 설정**
2. **무료 도메인 서비스 사용** (예: Freenom)
3. **DNS 연결**

## 📋 **애드센스 승인을 위한 체크리스트**

### **✅ 필수 요구사항**
- [ ] **HTTPS 보안**: 사이트가 HTTPS로 접근 가능
- [ ] **반응형 디자인**: 모바일에서도 정상 표시
- [ ] **로딩 속도**: 3초 이내 로딩
- [ ] **콘텐츠 품질**: 고품질의 독창적인 콘텐츠
- [ ] **사이트 연령**: 최소 6개월 이상 운영
- [ ] **트래픽**: 월 10,000명 이상 방문자

### **✅ 기술적 요구사항**
- [ ] **애드센스 코드 삽입**: `<head>` 태그 내에 정확히 삽입
- [ ] **광고 위치 설정**: 실제 광고가 표시될 위치 지정
- [ ] **robots.txt**: 검색엔진 크롤링 허용
- [ ] **sitemap.xml**: 사이트맵 제공

### **✅ 정책 준수**
- [ ] **저작권 준수**: 원본 콘텐츠 또는 적절한 인용
- [ ] **적절한 콘텐츠**: 성인 콘텐츠, 폭력 등 금지
- [ ] **사용자 경험**: 과도한 광고 배치 금지
- [ ] **개인정보 보호**: 개인정보처리방침 제공

## 🚀 **빠른 승인을 위한 팁**

### **1. 콘텐츠 품질 향상**
- **독창적인 콘텐츠**: 복사/붙여넣기가 아닌 원본 작성
- **정기 업데이트**: 주 2-3회 이상 콘텐츠 업데이트
- **사용자 참여**: 댓글, 공유 기능 활성화

### **2. 기술적 최적화**
- **페이지 속도**: 이미지 최적화, 코드 압축
- **모바일 최적화**: 모든 기기에서 정상 작동
- **SEO 최적화**: 메타 태그, 제목 태그 최적화

### **3. 사용자 경험 개선**
- **직관적인 네비게이션**: 사용자가 쉽게 찾을 수 있는 구조
- **빠른 로딩**: 3초 이내 페이지 로딩
- **오류 없는 페이지**: 404, 500 에러 최소화
