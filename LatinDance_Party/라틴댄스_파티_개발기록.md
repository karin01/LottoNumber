# 라틴댄스 파티 홍보 웹앱 개발 기록

## 📋 프로젝트 개요
라틴댄스 파티 정보를 공유하고 홍보할 수 있는 웹 애플리케이션입니다.

### 🎯 주요 기능
- 파티 정보 등록/수정/삭제
- 파티 목록 조회 및 필터링
- 포스터 이미지 업로드
- 좋아요 기능
- 댓글 시스템
- 유튜브 영상 갤러리
- 구글 로그인 연동
- 권한 관리 시스템

### 🔧 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Python Flask
- **Database**: Firebase Firestore
- **Storage**: Firebase Storage
- **Authentication**: Firebase Auth (Google Sign-In)
- **Hosting**: Firebase Hosting

#### 🔧 문제 상황
- 손학생이 관리자 계정이 아닌데도 모든 파티를 수정할 수 있는 문제 발생
- 권한 확인 로직이 UI에 제대로 반영되지 않음
- 수정/삭제 버튼이 권한과 관계없이 모든 사용자에게 표시됨

#### ✅ 해결된 문제들
1. **권한 없는 사용자 수정/삭제 방지**: 관리자와 등록자만 수정/삭제 가능

#### 🎯 권한 체계 정리
- **관리자 계정** (`jungwon1023@gmail.com`): 모든 파티 수정/삭제 가능
- **일반 사용자**: 자신이 등록한 파티만 수정/삭제 가능
- **비로그인 사용자**: 파티 목록 조회만 가능, 수정/삭제 버튼 표시되지 않음

1. 관리자 계정으로 로그인 → 모든 파티에 수정/삭제 버튼 표시 확인
2. 일반 사용자로 로그인 → 자신이 등록한 파티에만 수정/삭제 버튼 표시 확인

4. 권한 없는 파티 수정/삭제 시도 → 적절한 오류 메시지 표시

#### 1. 유튜브 URL 파싱
```javascript
// 유튜브 URL에서 비디오 ID 추출
function extractYouTubeVideoId(url) {
    const patterns = [
        /(?:youtube\.com\/watch\?v=|youtu\.be\/|youtube\.com\/embed\/)([^&\n?#]+)/,
        /youtube\.com\/watch\?.*v=([^&\n?#]+)/
    ];
    
    for (const pattern of patterns) {
        const match = url.match(pattern);
        if (match) {
            return match[1];
        }
    }
    
    return null;
}
```

#### 2. 영상 등록 처리
```javascript
// 유튜브 영상 등록 처리
async function handleYouTubeSubmit(event) {
    event.preventDefault();
    
    const title = document.getElementById('youtube-title').value.trim();
    const url = document.getElementById('youtube-url').value.trim();
    const description = document.getElementById('youtube-description').value.trim();
    const category = document.getElementById('youtube-category').value;
    const uploader = document.getElementById('youtube-uploader').value.trim();
    
    // 유효성 검사
    if (!title || !url || !category) {
        showMessage('제목, 유튜브 링크, 카테고리는 필수 입력 항목입니다.', 'error');
        return;
    }
    
    // 유튜브 URL에서 비디오 ID 추출
    const videoId = extractYouTubeVideoId(url);
    if (!videoId) {
        showMessage('올바른 유튜브 링크를 입력해주세요.', 'error');
        return;
    }
    
    // 새 영상 객체 생성
    const newVideo = {
        id: Date.now().toString(),
        title: title,
        url: url,
        videoId: videoId,
        description: description,
        category: category,
        uploader: uploader || '익명',
        thumbnail: `https://img.youtube.com/vi/${videoId}/maxresdefault.jpg`,
        createdAt: new Date().toISOString()
    };
    
    // Firebase에 저장
    await saveYouTubeVideo(newVideo);
    
    // 폼 초기화 및 목록 새로고침
    document.getElementById('youtube-form').reset();
    await loadYouTubeVideos();
    
    showMessage('영상이 성공적으로 등록되었습니다!', 'success');
}
```

```javascript
// 유튜브 영상 목록 표시
function displayYouTubeVideos() {
    const youtubeContainer = document.getElementById('youtube-container');
    const categoryFilter = document.getElementById('youtube-category-filter');
    const searchInput = document.getElementById('youtube-search');
    
    // 필터링
    let filteredVideos = [...youtubeVideos];
    
    // 카테고리 필터
    if (categoryFilter && categoryFilter.value) {
        filteredVideos = filteredVideos.filter(video => video.category === categoryFilter.value);
    }
    
    // 검색 필터
    if (searchInput && searchInput.value.trim()) {
        const searchTerm = searchInput.value.trim().toLowerCase();
        filteredVideos = filteredVideos.filter(video => 
            video.title.toLowerCase().includes(searchTerm) ||
            video.description.toLowerCase().includes(searchTerm) ||
            video.uploader.toLowerCase().includes(searchTerm)
        );
    }
    
    // 최신순으로 정렬
    filteredVideos.sort((a, b) => new Date(b.createdAt) - new Date(a.createdAt));
    
    // 영상 카드 생성
    youtubeContainer.innerHTML = filteredVideos.map(video => `
        <div class="youtube-card" onclick="openYouTubeModal('${video.id}')">
            <img src="${video.thumbnail}" alt="${video.title}" class="youtube-thumbnail" 
                 onerror="this.src='https://via.placeholder.com/320x180/ff6b6b/ffffff?text=라틴댄스'">
            <div class="youtube-info">
                <div class="youtube-title">${escapeHtml(video.title)}</div>
                <div class="youtube-channel">${escapeHtml(video.uploader)}</div>
                <div class="youtube-category">${video.category}</div>
                ${video.description ? `<div class="youtube-description">${escapeHtml(video.description)}</div>` : ''}
                <div class="youtube-duration">${formatDate(video.createdAt)}</div>
            </div>
        </div>
    `).join('');
}
```

## 2025-01-13 - 파티 등록 에러 해결

### 문제 상황
- 파티 등록 시 `FirebaseError: The value of property "posterUrl" is longer than 1048487 bytes` 에러 발생
- Firestore의 각 필드 1MB 제한을 초과하는 문제

### 원인 분석
- 포스터 이미지를 Base64로 변환하여 Firestore에 직접 저장하고 있었음
- Base64 데이터는 이미지 크기에 따라 수 MB까지 커질 수 있어 Firestore 제한 초과

### 해결 방법

#### 1. 프론트엔드 수정 (main.js)
```javascript
// 기존: Base64 변환 후 저장 (❌)
const posterUrl = await new Promise((resolve, reject) => {
    const reader = new FileReader();
    reader.onload = function(e) {
        resolve(e.target.result); // data:image/png;base64,...
    };
    reader.readAsDataURL(posterFile);
});

// 수정: Firebase Storage 업로드 후 URL 저장 (✅)
const posterUrl = await uploadPoster(posterFile); // https://firebasestorage.googleapis.com/...
```

#### 2. 파일 크기 제한 추가
```javascript
// 1MB 제한 체크
const maxSize = 1024 * 1024;
if (posterFile.size > maxSize) {
    throw new Error('포스터 파일 크기가 1MB를 초과합니다.');
}
```

#### 3. 백엔드 보호 장치 추가 (app.py)
```python
# Base64 데이터 방지
if poster_url.startswith('data:image'):
    return jsonify({"result": "error", "message": "Base64 데이터는 저장할 수 없습니다."}), 400

# URL 길이 체크
if len(poster_url.encode('utf-8')) > 1048487:
    return jsonify({"result": "error", "message": "포스터 URL이 너무 깁니다."}), 400
```

- `LatinDance_Party/static/js/main.js`: 포스터 업로드 로직 수정
- `LatinDance_Party/backend/app.py`: 백엔드 검증 로직 추가

- 영어 버튼을 눌러도 한글로 그대로 표시됨
- 번역 API 호출이 제대로 작동하지 않음
- UI 요소들이 영어로 변경되지 않음

### 해결 방법

#### 1. 직접적인 UI 변경 방식으로 수정
기존의 번역 API 호출 방식 대신 직접적인 DOM 조작으로 변경:

```javascript
// 헤더 변경
document.querySelector('header h1').textContent = '🕺💃 Latin Dance Party Community';
document.querySelector('header p').textContent = 'Discover Latin dance parties nationwide and meet new friends!';

// 파티 카드 정보 변경
const infoRows = card.querySelectorAll('.party-info');
infoRows.forEach(row => {
    const text = row.textContent;
    if (text.includes('지역:')) {
        row.innerHTML = row.innerHTML.replace('지역:', 'Region:');
    } else if (text.includes('바 이름:')) {
        row.innerHTML = row.innerHTML.replace('바 이름:', 'Bar:');
    }
    // ... 기타 필드들
});
```

#### 2. 한국어 복원 기능 추가
```javascript
function toggleKorean() {
```

#### 3. 번역 대상 요소 확장
- **헤더**: 제목과 설명
- **필터**: 라벨과 옵션
- **파티 카드**: 모든 정보 필드
- **푸터**: 저작권 정보
- **버튼**: 타이틀 버튼들

- `LatinDance_Party/static/js/main.js`: `toggleEnglish()`, `toggleKorean()` 함수 수정

### 결과

- **완전한 번역**: 모든 UI 요소가 영어/한국어로 변경
- **안정성**: 번역 API 의존성 제거로 안정적 작동

## 2025-01-XX - 스페인어 번역 기능 완성

### 문제 상황
- ESP 버튼이 제대로 작동하지 않음
- 파티 카드 정보가 스페인어로 변경되지 않음
- 언어 간 전환이 완전하지 않음

### 해결 방법

#### 1. 스페인어 번역 기능 완성
```javascript
// 파티 카드 스페인어 번역
const partyCards = document.querySelectorAll('.party-card');
partyCards.forEach(card => {
    const infoRows = card.querySelectorAll('.party-info');
    infoRows.forEach(row => {
        const text = row.textContent;
        if (text.includes('지역:') || text.includes('Region:')) {
            row.innerHTML = row.innerHTML.replace('지역:', 'Región:').replace('Region:', 'Región:');
        } else if (text.includes('바 이름:') || text.includes('Bar:')) {
            row.innerHTML = row.innerHTML.replace('바 이름:', 'Bar:').replace('Bar:', 'Bar:');
        } else if (text.includes('상세주소:') || text.includes('Address:')) {
            row.innerHTML = row.innerHTML.replace('상세주소:', 'Dirección:').replace('Address:', 'Dirección:');
        }
        // ... 기타 필드들
    });
});
```

#### 2. 크로스 언어 전환 지원
- **한국어 → 영어**: 모든 필드 영어로 변경
- **한국어 → 스페인어**: 모든 필드 스페인어로 변경
- **영어 → 한국어**: 모든 필드 한국어로 복원
- **영어 → 스페인어**: 모든 필드 스페인어로 변경
- **스페인어 → 한국어**: 모든 필드 한국어로 복원
- **스페인어 → 영어**: 모든 필드 영어로 변경

#### 3. 스페인어 번역 대상
- **헤더**: "Comunidad de Fiestas de Baile Latino"
- **설명**: "¡Descubre fiestas de baile latino en todo el país y conoce nuevos amigos!"
- **필터**: "Región:", "Fecha:", "Todas las Regiones"
- **파티 카드**:
  - "지역:" → "Región:"
  - "바 이름:" → "Bar:"
  - "상세주소:" → "Dirección:"
  - "장소:" → "Lugar:"
  - "일시:" → "Fecha:"
  - "등록자:" → "Registrante:"
- **버튼**: "Registrar Fiesta", "Ver Álbum", "Galería de Videos"

### 수정된 파일
- `LatinDance_Party/static/js/main.js`: `toggleSpanish()`, `toggleEnglish()`, `toggleKorean()` 함수 수정

### 결과
- **완전한 3개 언어 지원**: 한국어, 영어, 스페인어
- **자유로운 언어 전환**: 어떤 언어에서든 다른 언어로 즉시 전환
- **일관된 사용자 경험**: 모든 UI 요소가 선택된 언어로 변경
- **안정적인 작동**: 번역 API 없이 직접 DOM 조작으로 안정적

### 참고사항
- Firebase Storage는 이미지 파일 저장에 최적화되어 있음
- Firestore는 메타데이터 저장에만 사용하는 것이 좋음
- 이미지 URL은 보통 수백 바이트 정도로 매우 작음
