# 카카오톡 공유 기능 설정 가이드

## 🎯 목표
파티 카드를 카카오톡에 공유할 때 이미지와 함께 링크가 표시되고, 링크를 클릭하면 바로 파티 상세 정보 모달이 열리도록 설정합니다.

## 📋 설정 단계

### 1. 카카오 개발자 계정 생성
1. [Kakao Developers](https://developers.kakao.com/) 접속
2. 카카오 계정으로 로그인
3. "내 애플리케이션" → "애플리케이션 추가하기" 클릭

### 2. 애플리케이션 등록
1. **앱 이름**: "라틴댄스 파티" (또는 원하는 이름)
2. **사업자명**: 개인 또는 회사명
3. **저장** 버튼 클릭

### 3. 플랫폼 설정
1. **플랫폼** → **Web** 선택
2. **사이트 도메인**에 다음 추가:
   - `http://localhost:5000` (개발용)
   - `https://your-domain.com` (실제 도메인)
3. **저장** 버튼 클릭

### 4. JavaScript 키 복사
1. **앱 키** 섹션에서 **JavaScript 키** 복사
2. 예시: `1234567890abcdef1234567890abcdef`

### 5. 코드에 키 적용
`LatinDance_Party/static/js/main.js` 파일의 3415번째 줄을 수정:

```javascript
// 변경 전
const kakaoKey = 'YOUR_KAKAO_JAVASCRIPT_KEY';

// 변경 후 (실제 키로 교체)
const kakaoKey = '1234567890abcdef1234567890abcdef';
```

### 6. 카카오톡 채널 추가 (선택사항)
더 나은 공유 경험을 위해 카카오톡 채널을 추가할 수 있습니다:
1. **카카오톡 채널** → **채널 추가하기**
2. 채널명과 설명 입력
3. 채널 ID를 복사하여 공유 데이터에 추가

## 🧪 테스트 방법

### 1. 로컬 테스트
```bash
# 서버 실행
cd LatinDance_Party
python app.py
```

### 2. 공유 테스트
1. 파티 카드의 "공유" 버튼 클릭
2. 카카오톡 앱에서 공유 확인
3. 공유된 링크 클릭하여 모달 열기 확인

### 3. 콘솔에서 링크 확인
브라우저 개발자 도구 콘솔에서:
```javascript
generateShareLinks()
```

## 🔧 문제 해결

### 카카오톡 공유가 작동하지 않는 경우
1. **앱 키 확인**: 올바른 JavaScript 키가 설정되었는지 확인
2. **도메인 확인**: 플랫폼 설정에 올바른 도메인이 등록되었는지 확인
3. **HTTPS 확인**: 실제 서비스에서는 HTTPS가 필요할 수 있음

### 이미지가 표시되지 않는 경우
1. **이미지 URL 확인**: 절대 경로로 설정되었는지 확인
2. **CORS 설정**: 이미지 서버에서 CORS 허용 필요
3. **이미지 크기**: 카카오톡 권장 크기 (400x300px 이상)

## 📱 모바일 최적화
