# HTML 줄바꿈 문제 해결 기록

## 🚨 문제 상황
- **사용자 피드백**: "CSS 다른거 수정되고 있는거 아냐??"
- **실제 문제**: HTML에서 직접 `<br>` 태그를 사용하여 강제 줄바꿈
- **영향**: CSS 스타일이 적용되지 않아 반응형 제목이 제대로 작동하지 않음

## 🔍 문제 진단

### 1. 잘못된 접근
- ❌ **CSS만 수정**: `.welcome-section h4` 스타일만 수정
- ❌ **캐시 문제로 오인**: 브라우저 캐시 문제로 잘못 진단
- ❌ **버전 파라미터 추가**: 불필요한 CSS 버전 관리

### 2. 실제 문제 발견
- ✅ **HTML 소스 확인**: `index.html`에서 직접 `<br>` 태그 사용
- ✅ **문제 위치**: `latin-dance-guide` 섹션의 `h2` 태그
- ✅ **원인**: HTML에서 강제 줄바꿈으로 CSS 스타일 무시

## 🔧 해결 방안

### 1. HTML 수정
**위치**: `LatinDance_Party/index.html` - 383번째 줄

**변경 사항**:
```html
<!-- 기존 (문제) -->
<h2>💃 라틴댄스<br>완전 가이드</h2>

<!-- 수정 후 (해결) -->
<h2>💃 라틴댄스 완전 가이드</h2>
```

### 2. CSS 스타일 추가
**위치**: `LatinDance_Party/static/css/components.css`

**추가된 스타일**:
```css
/* PC 버전 (기본) */
section h2 {
    color: #e74c3c;
    font-size: 2rem;
    margin-bottom: 1.5rem;
    text-align: center;
    position: relative;
    font-weight: 700;
    display: flex;           /* 추가: 플렉스 레이아웃 */
    align-items: center;     /* 추가: 세로 중앙 정렬 */
    justify-content: center; /* 추가: 가로 중앙 정렬 */
    gap: 0.5rem;            /* 추가: 요소 간 간격 */
    flex-wrap: nowrap;       /* 추가: 줄바꿈 방지 */
}

/* 모바일 버전 */
@media (max-width: 768px) {
    section h2 {
        flex-wrap: wrap;         /* 줄바꿈 허용 */
        flex-direction: column;  /* 세로 배치 */
        align-items: center;     /* 중앙 정렬 */
        gap: 0.3rem;            /* 간격 조정 */
    }
}
```

### 3. CSS 버전 업데이트
**위치**: `LatinDance_Party/index.html`

**변경 사항**:
```html
<!-- 기존 -->
<link rel="stylesheet" href="static/css/components.css?v=1.1">

<!-- 수정 후 -->
<link rel="stylesheet" href="static/css/components.css?v=1.2">
```

## 📋 해결 단계

### 1단계: HTML 수정
- ✅ `<br>` 태그 제거
- ✅ 제목을 한 줄로 통합
- ✅ CSS 스타일이 적용될 수 있도록 준비

### 2단계: CSS 스타일 추가
- ✅ `section h2`에 플렉스 레이아웃 적용
- ✅ PC 버전에서 한 줄 표시 설정
- ✅ 모바일 버전에서 줄바꿈 설정

### 3단계: 버전 업데이트
- ✅ CSS 파일 버전을 1.2로 업데이트
- ✅ 브라우저 캐시 무효화 유도

## 🎯 예상 결과

### PC 버전 (768px 이상)
- **레이아웃**: 가로 배치 (flex-direction: row)
- **줄바꿈**: 없음 (flex-wrap: nowrap)
- **정렬**: 중앙 정렬
- **결과**: "💃 라틴댄스 완전 가이드" 한 줄 표시

### 모바일 버전 (768px 미만)
- **레이아웃**: 세로 배치 (flex-direction: column)
- **줄바꿈**: 허용 (flex-wrap: wrap)
- **정렬**: 중앙 정렬
- **결과**: 
  - "💃 라틴댄스" (첫 줄)
  - "완전 가이드" (둘째 줄)

## 📊 기술적 세부사항

### Flexbox 활용
- **display: flex**: 플렉스 레이아웃 적용
- **align-items**: 세로 정렬 제어
- **justify-content**: 가로 정렬 제어
- **gap**: 요소 간 간격 설정
- **flex-wrap**: 줄바꿈 제어
- **flex-direction**: 배치 방향 제어

### 미디어 쿼리
- **브레이크포인트**: 768px
- **적용 범위**: 태블릿 및 모바일
- **우선순위**: 모바일 스타일이 PC 스타일을 덮어씀
